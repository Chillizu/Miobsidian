# 第三章大纲 — 时间复杂度与安全复杂度

> **证明目标**：在 §2 的群结构上逐一拆解核心运算步骤、估代价、求总复杂度，并分析对应的最有效攻击复杂度。每节以"如第二章所述"开头。

---

## 3.1 时间复杂度

### 3.1.1 RSA
- **已知（§2.1.1）**：$(\mathbb{Z}/n\mathbb{Z})^\times$ 上模幂 $c^d \bmod n$，$d \approx n$
- **推导步骤**：
  1. $d$ 二进制展开：$d = \sum b_i 2^i$，$k = \log_2 d$ 位
  2. MSB 扫描：每步平方 $+$ 条件乘 → $\frac{3}{2}k$ 次模乘
  3. 单次模乘：$b$ 位整数乘 $O(b^2)$，$b = \log_2 n$
- **结果**：$T_{\text{RSA}} = O(\log n) \times O((\log n)^2) = O((\log n)^3)$
- **加密 vs 解密**：$e=65537$（$17$ 位）→ 迭代数极小，虽同量级但实际远快
- **引用**：[15]

### 3.1.2 DH
- **已知（§2.1.2）**：$\langle g \rangle \subset (\mathbb{Z}/p\mathbb{Z})^\times$，双方各算 $g^a \bmod p$
- **推导**：同 RSA 平方-乘法 → $O(\log^3 p)$
- **引用**：[14]

### 3.1.3 ElGamal
- **已知（§2.1.3）**：加密需 $g^y$ 与 $h^y$ 两次模幂，解密需 $c_1^{-x}$ 一次
- **推导**：加密 $2\times T_{\text{DH}}$，解密 $1\times T_{\text{DH}}$ → 均为 $O(\log^3 p)$
- **引用**：[2]

### 3.1.4 DSA
- **已知（§2.1.4）**：指数 $k, u_1, u_2$ 取自 $\mathbb{F}_q$（$256$ bits），模数 $p$（$2048$ bits）
- **推导**：
  1. 模幂迭代次数 $O(\log q)$
  2. 单次模乘 $O((\log p)^2)$
  3. 总 $O(\log q \cdot (\log p)^2)$
- **结果**：$\log q = O(\log p)$ → 综合量级 $O((\log p)^3)$
- **引用**：[10]

### 3.1.5 ECC
- **已知（§2.1.5）**：$E(\mathbb{F}_q)$ 上标量乘 $Q = kP$，$k \approx 2^{256}$
- **推导**：
  1. 倍点-加法（MSB 扫描）→ $\frac{3}{2}\log_2 k$ 次点运算，$O(\log q)$
  2. Weierstrass 仿射坐标：点加（$2$ 域乘 $+ 1$ 域逆），倍点（$3$ 域乘 $+ 1$ 域逆）
  3. 域乘/域逆均 $O((\log q)^2)$
- **结果**：$T_{\text{ECC}} = O(\log^3 q)$
- **与 RSA 对比**：$256$-bit ECC vs $3072$-bit RSA，等比安全下快 $5$–$10$ 倍
- **引用**：[16][17][4]

### 3.1.6 AES
- **已知（§2.1.6）**：10/12/14 轮 SPN，每轮 SubBytes + ShiftRows + MixColumns + AddRoundKey
- **推导**：轮数由密钥长固定，每轮操作数固定 → 每块 $O(1)$，$n$ 块 $O(n)$
- **引用**：[13]

### 3.1.7 DES / 3DES
- **已知（§2.2）**：16 轮 Feistel，$F$ 内全为位操作和 S-box 查表
- **推导**：每块操作数固定 → 每块 $O(1)$，$n$ 块 $O(n)$
- **引用**：[1]

### 3.1.8 MD5 / SHA-1 / SHA-256
- **已知（§2.3）**：Merkle–Damgård，固定步数
- **推导**：每块 $O(1)$，$n$ 块 $O(n)$
- **引用**：[1][7]

### 3.1.9 复杂度对比表

| 算法 | 复杂度 | 类型 |
|:---|---:|:---|
| RSA | $O(\log^3 n)$ | 三次多项式 |
| DH | $O(\log^3 p)$ | 三次多项式 |
| ElGamal | $O(\log^3 p)$ | 三次多项式（2-3×） |
| DSA | $O(\log q \cdot (\log p)^2)$ | 三次多项式 |
| ECC | $O(\log^3 q)$ | 三次多项式（常数远小） |
| AES/DES/Hash | $O(n)$ | 线性 |

## 3.2 安全复杂度

### 3.2.1 RSA
- **归约**：大整数分解
- **攻击**：GNFS $L_n[1/3, (64/9)^{1/3}] \approx \exp(1.923 (\ln n)^{1/3} (\ln \ln n)^{2/3})$
- **等效安全**：RSA-2048 → $\sim 112$ bits，RSA-3072 → $128$ bits
- **引用**：[8][6]

### 3.2.2 DH / ElGamal / DSA
- **归约**：$(\mathbb{Z}/p\mathbb{Z})^\times$ 上的 DLP
- **攻击**：GNFS 变体 $L_p[1/3, 1.923]$
- **引用**：[9]

### 3.2.3 ECC
- **归约**：$E(\mathbb{F}_q)$ 上的 ECDLP
- **攻击**：
  - 一般曲线：Pollard-$\rho$ $O(\sqrt{q})$（指数级）
  - 超奇异曲线：MOV 攻击降为 $\mathbb{F}_{q^k}^\times$ 上的 DLP → GNFS 亚指数
  - 异常曲线（$\#E = q$）：Smart 攻击 $O(\log^3 q)$（多项式）→ 标准曲线均规避
- **等效安全**：$256$-bit 曲线 → $128$ bits
- **引用**：[12][23][3]

### 3.2.4 AES
- **穷举**：$O(2^k)$（$k$ = 密钥长）
- **Biclique 攻击**：$2^{126.1}$（$k=128$），$2^{254.4}$（$k=256$）
- **量子**：Grover 算法 $O(2^{k/2})$
- **引用**：[11][20]

### 3.2.5 DES
- **穷举**：$2^{56}$，EFF Deep Crack 已实现
- **引用**：[22]

### 3.2.6 MD5 / SHA-1 / SHA-256
- **MD5**：碰撞 $2^{39}$（Wang & Yu, EUROCRYPT 2005）
- **SHA-1**：碰撞 $2^{63}$（SHAttered 2017）
- **SHA-256**：碰撞 $2^{128}$（穷举原像 $2^{256}$），安全
- **引用**：[21][7]

### 3.2.7 安全复杂度对比表

| 算法 | 攻击模型 | 代价 | 等效安全 (bits) | 状态 |
|:---|:---|:---|---:|:---:|
| RSA-2048 | GNFS | $L_n[1/3, 1.923]$ | $\sim 112$ | 临期 |
| RSA-3072 | GNFS | $L_n[1/3, 1.923]$ | $128$ | 安全 |
| DH/DSA-2048 | GNFS-DLP | $L_p[1/3, 1.923]$ | $\sim 112$ | 临期 |
| ECC-256 | Pollard-$\rho$ | $O(2^{128})$ | $128$ | 安全 |
| AES-128 | Biclique | $2^{126.1}$ | $126$ | 安全 |
| AES-256 | 穷举 | $2^{256}$ | $256$ | 安全 |
| DES-56 | 穷举 | $2^{56}$ | $56$ | 已破 |
| MD5 | 碰撞攻击 | $2^{39}$ | — | 已破 |
| SHA-1 | 碰撞攻击 | $2^{63}$ | — | 已破 |
| SHA-256 | 碰撞 | $2^{128}$ | $128$ | 安全 |
