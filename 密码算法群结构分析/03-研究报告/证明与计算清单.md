# 证明与计算清单

> 本文需数学证明或计算推导的所有条目，按节排列。每个条目标注性质：[证明] = 需代数推导，[计算] = 需复杂度分析，[验证] = 需实验或数值验证。

## §1.1 核心数学问题

### §1.1.1 阿贝尔群

- **[证明]** DLP ⇒ DHP：若已知 $a = \log_g h$，则 $g^{ab} = (g^b)^a$ → 解 DHP。方向：给定 $g^a, g^b$ 求 $g^{ab}$，若已知 $a$ 则直接计算。
- **[证明]** $(\mathbb{Z}/p\mathbb{Z})^\times$ 为循环群：引用原根定理——仅陈述，不做完整证明，注明"见 [3] ch.1"。

### §1.1.2 非阿贝尔群

- **[证明]** 辫群 $B_n$ 的关系式 $\sigma_i\sigma_{i+1}\sigma_i = \sigma_{i+1}\sigma_i\sigma_{i+1}$：定义陈述，无需证明。
- **[证明]** CSP 在阿贝尔群中平凡：若 $G$ 为 Abel，则 $g^{-1}xg = x$ 恒成立，CSP 无意义。仅陈述。

## §1.2 基于群的加密算法

### §1.2.1 RSA

- **[证明]** 解密正确性：$c^d \equiv (m^e)^d = m^{ed} = m^{1+t\varphi(n)} \equiv m \pmod{n}$。前提：$ed \equiv 1 \pmod{\varphi(n)}$，Euler 定理 $m^{\varphi(n)} \equiv 1 \pmod{n}$。
- **[证明]** $\varphi(n) = (p-1)(q-1)$：由 $n=pq$ 直接计算（Euler 函数的乘性 + 素数的 Euler 函数值）。可略证。
- **[证明]** 签名模式正确性：$s^e \equiv (H(m)^d)^e = H(m)^{ed} \equiv H(m) \pmod{n}$。与加密证明相同，仅方向互换。

### §1.2.2 DH / ElGamal

- **[证明]** DH 密钥一致性：$A^b = (g^a)^b = g^{ab} = (g^b)^a = B^a$。直接由指数乘法交换律。
- **[证明]** ElGamal 解密正确性：$c_2 \cdot (c_1^x)^{-1} = m \cdot h^y \cdot (g^{xy})^{-1} = m \cdot g^{xy} \cdot g^{-xy} = m$。前提：$h = g^x$，$c_1 = g^y$。

### §1.2.3 DSA

- **[证明]** DSA 验证正确性：展开 $g^{u_1} y^{u_2} = g^{H(m)s^{-1}} \cdot g^{xrs^{-1}} = g^{(H(m)+xr)s^{-1}}$。由 $s = k^{-1}(H(m)+xr) \bmod q$ 得 $s^{-1} \equiv k(H(m)+xr)^{-1} \pmod{q}$，代入得 $g^k \bmod p$，模 $q$ 即 $r$。
- **[证明]** $k$ 复用泄露私钥：若 $(r, s_1)$ 和 $(r, s_2)$ 使用相同 $k$，则 $s_1 - s_2 = k^{-1}(H(m_1) - H(m_2)) \bmod q$ → $k = (H(m_1) - H(m_2))(s_1 - s_2)^{-1} \bmod q$ → $x = r^{-1}(ks_1 - H(m_1)) \bmod q$。

### §1.2.4 ECC

- **[证明]** 点加法公式推导（Weierstrass 曲线特征 ≠ 2,3）：
  - 弦斜率 $\lambda = (y_2 - y_1)(x_2 - x_1)^{-1}$
  - 代入曲线：$(\lambda(x - x_1) + y_1)^2 = x^3 + ax + b$
  - 整理为三次方程 → Viète 公式 $x_1 + x_2 + x_3' = \lambda^2$ → $x_3' = \lambda^2 - x_1 - x_2$
  - 反射：$x_3 = x_3'$，$y_3 = \lambda(x_1 - x_3) - y_1$
- **[证明]** 倍点公式推导：切线斜率 $\lambda = (3x_1^2 + a)(2y_1)^{-1}$，其余同加法。需给出切线方程的求导依据。
- **[证明]** Hasse 不等式：$| |E(\mathbb{F}_q)| - (q+1) | \leq 2\sqrt{q}$。仅陈述——完整证明过长（引用 [5] ch.3）。

### §1.2.6 AES

- **[证明]** $\mathrm{GF}(2^8)$ 的域结构：$x^8+x^4+x^3+x+1$ 不可约性。仅陈述——引用 AES 规范 [9]。
- **[证明]** S-box 无不动点性质：$\forall x, S(x) \neq x$。由仿射矩阵 $A$ 的构造保证。可略证，引用 [9] §3.4。
- **[计算]** MixColumns 分支数 = 5：对矩阵每列，改变 $k$ 字节导致输出至少 $5-k$ 字节改变。引用 MDS 矩阵性质。

### §1.2.7 MD4 / MD5 / SHA

- 无群论证明需求——Merkle–Damgård 迭代构造仅作描述性陈述。

## §2.1 时间复杂度

### RSA

- **[计算]** 平方-乘法迭代数：$d = \sum b_i 2^i$，$k = \lfloor \log_2 d \rfloor + 1$。MSB 扫描：$k-1$ 次平方 $+ \sim (k-1)/2$ 次条件乘法。总计 $\frac{3}{2}k + O(1)$ 次模乘。
- **[计算]** 单次模乘代价：$b$ 位整数乘 $O(b^2)$（教科书乘法），$b = \log_2 n$ → $O((\log n)^2)$。模约简同量级。
- **[计算]** 加密特例 $e = 65537 = 2^{16}+1$：$16$ 次平方 + $1$ 次条件乘 → $O((\log n)^2)$（渐近同量级但实际远快）。

### DH

- **[计算]** 同 RSA——双方各一次平方-乘法。无额外推导需求。

### ElGamal

- **[计算]** 加密 = $g^y$, $h^y$ 两次模幂 + $m \cdot h^y$。解密 = $(c_1^x)^{-1}$ 一次模幂。总计 $2$–$3$ 次模幂。

### DSA

- **[计算]** 指数位长 $\log q$ vs 模数位长 $\log p$。迭代数 $O(\log q)$，单次模乘 $O((\log p)^2)$。总 $O(\log q \cdot (\log p)^2)$。

### ECC

- **[计算]** 倍点-加法迭代数：同平方-乘法——$\frac{3}{2}\log_2 k$ 次点运算。
- **[计算]** 单次点加/倍点代价（仿射坐标）：
  - 点加：$1$ 域逆 + $2$ 域乘
  - 倍点：$1$ 域逆 + $3$ 域乘
- **[计算]** 域运算是 $O((\log q)^2)$：乘法 $O((\log q)^2)$（教科书），逆通过扩展欧几里得 $O((\log q)^2)$。
- **[计算]** ECC vs RSA 常数因子对比：$(\frac{3072}{256})^3 \approx 1728$，部分被 $e=65537$ 和射影坐标抵消 → 实际约快 $5$–$10$ 倍。

## §2.2 安全复杂度

### RSA / DH / DSA

- **[计算]** GNFS 复杂度 $L_n[1/3, (64/9)^{1/3}]$。仅陈述——$L$ 函数的定义和 $\alpha=1/3$ 的意义各用 1 句解释。
- **[计算]** 等效安全比特：设 GNFS 攻击代价 ≈ $2^{112}$ 次操作 → RSA-2048 = 112 bits。仅陈述，引用 [6] NIST SP 800-57。

### ECC

- **[计算]** Pollard-$\rho$ 复杂度 $O(\sqrt{q})$。核心推理：生日悖论 → 在 $N$ 元素集合中随机选取 $M \approx \sqrt{N}$ 个元素，碰撞概率 $\approx 1/2$。$\sqrt{q} \approx 2^{128}$ 对 $q \approx 2^{256}$。
- **[计算]** MOV 攻击条件：超奇异曲线嵌入次数 $k \leq 6$ → ECDLP 降为 $\mathbb{F}_{q^k}^\times$ 上的 DLP → GNFS 亚指数。仅陈述，引用 [21]。
- **[计算]** Smart 攻击条件：$|E(\mathbb{F}_q)| = q$（异常曲线）→ $O(\log^3 q)$ 多项式时间。仅陈述，引用 [3]。

### AES

- **[计算]** 穷举 $O(2^k)$（$k=128, 192, 256$）。Biclique 降至 $2^{126.1}$。Grover 量子降至 $O(2^{k/2})$。

### 哈希

- **[计算]** 生日攻击：$n$ 位输出的碰撞期望 $\approx 2^{n/2}$ 次尝试。基于生日悖论推导——随机选取 $M$ 个元素，任意两元素相等的概率 $P \approx 1 - e^{-M^2/(2N)}$。当 $M = \sqrt{N}$ 时 $P \approx 1 - e^{-1/2} \approx 0.39$。
- **[计算]** MD5 $2^{39}$ / SHA-1 $2^{63}$：引用 [15] 和 SHAttered（2017）的具体数字，不推导攻击方法。

## §3 实验测试

### §3.1 哈希碰撞

- **[验证]** MD4 碰撞：使用公开工具生成 → 展示 $H(M_1) = H(M_2)$。碰撞消息对引用已知结果，不要求自己发现。
- **[验证]** SHA-1 碰撞：引用 SHAttered（2017）PDF 碰撞。提供碰撞文件的 SHA-1 哈希相等性。

### §3.2 非对称参数测试

- **[验证]** 大步小步法解小素数 DLP：$m = \lceil \sqrt{p} \rceil$，Baby Steps $\{ (g^j, j) \}_{j=0}^{m-1}$，Giant Steps 匹配 → $a$ 恢复。以 $p \approx 10^5$ 为例演示数值。
- **[验证]** 奇异曲线 ECDLP：$\Delta = 0$ → 曲线同构于加法群或乘法群 → DLP 平凡化。以 $y^2 = x^3$ 为例演示。
- **[验证]** 异常曲线 Smart 攻击：$p$-adic 椭圆曲线提升。仅陈述结果（"可在线性时间内求解"），不推导 Smart 攻击。
