# 第三章章节大纲与说明

> 对应论文第三章。纯粹的算法分类陈述，不做复杂度推导。

---

## 分类

| 节 | 算法 | 群结构 |
|:---|:---|:---|
| 3.1.1 | RSA | $(\mathbb{Z}/n\mathbb{Z})^\times$ |
| 3.1.2 | DH | $\langle g \rangle \subset (\mathbb{Z}/p\mathbb{Z})^\times$ |
| 3.1.3 | ElGamal | 同 DH |
| 3.1.4 | DSA | 同 DH |
| 3.1.5 | ECC | $E(\mathbb{F}_q)$ |
| 3.1.6 | AES | $(\mathrm{GF}(2^8), +)$ 与 $(\mathrm{GF}(2^8)^\times, \cdot)$ |
| 3.2.1 | DES | 置换集（已证明非群） |
| 3.2.2 | 3DES | 级联置换（非群） |
| 3.3.1 | MD5 | 无群结构 |
| 3.3.2 | SHA-1 | 无群结构 |
| 3.3.3 | SHA-256 | 无群结构 |

## 每节模板

- 简介：算法是什么/谁设计的/安全依据
- 公式：核心计算公式
- 群结构：群类型和关键性质（仅陈述，不讨论复杂度或深层安全含义）
- 领域：主要应用
