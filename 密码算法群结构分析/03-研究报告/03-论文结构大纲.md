## 论文整体结构（对应 `Cipher Outline 1.docx`）

```
1) Introduction (~150 words)
   引入群论是密码学的安全基础设施，给出路线图。

2) Section 1: Cryptographic Algorithms (~2000 words)
   ├── a) The Principles
   │   ├── 1) Cyclic Groups: DHP, DLP
   │   └── 2) Non-Abelian Groups: CSP, Ko-Lee, AAG, Stickel
   ├── b) Group-based Algorithms
   │   ├── 1) Asymmetric: RSA, ElGamal, DSA
   │   └── 2) Symmetric: PGM
   ├── c) More Algorithms
   │   ├── 1) Encryption: AES
   │   ├── 2) Signature: ECC
   │   └── 3) Hash Functions: MD4-MD5, SHA-1-SHA-256
   └── d) More Cryptographic Practices
       ├── 1) Padding
       └── 2) KDF

3) Section 2: Properties and Applications (~600 words)
   ├── a) Properties (复杂度 + 等效安全表)
   ├── b) Applications (各算法用途 + postquantum.com)
   └── c) Cryptanalysis (经典攻击 + 量子威胁)

4) Section 3: Testing (~600 words)
   ├── a) Hash Collisions
   │   ├── 1) MD4 collision (Wang 2005)
   │   └── 2) SHA-1 collision (SHAttered 2017)
   └── b) Inappropriate Parameters
       ├── 1) Small-prime DLP (BSGS / Pohlig-Hellman)
       └── 2) Anomalous ECC curve (Smart p-adic lift)

5) Conclusion (~150 words)
   总结 + 量子威胁 + 后量子密码展望。
```
